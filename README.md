# 项目简介

## 核心功能
- 基于YOLOv8的鱼病害检测，支持5类识别：细菌性疾病(Bacterial Diseases)、真菌性疾病(Fungal Diseases)、健康鱼类(Healthy Fish)、寄生虫病(Parasitic Diseases)、白尾病(White Tail Diseases)
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 实时显示检测结果：目标位置、目标总数、置信度等信息
- 批量检测功能，可直接查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 可选择显示标签和原图
- 支持选择不同检测模型

## 适用场景
- 水产养殖病害监测
- 鱼类健康检查
- 水产研究机构数据分析
- 养殖场自动化检测系统

## 技术特点
- 开发环境：Pycharm+Anaconda 或 VSCode+Anaconda
- 编程语言：Python 3.9
- 主要依赖库：opencv-python, PyQt5, ultralytics, torch1.9
- 内置2500张标注数据集
- 提供完整源码、UI界面源文件、数据集图片和项目运行教程
- 配置成功环境可直接使用
